Помещаем в папку "EUBANK" папку "chromium" содержащую необходимые файлы, которые можно выкачать из http://security.ubuntu.com/ubuntu/pool/universe/c/chromium-browser/
у меня это целый набор, но можно ограничится конкретными 
3-мя файлами(chromium-browser_79*, chromium-codecs-ffmpeg-extra_79*, chromium-browser-l10n_79*) 
для Вашего дистрибутива
